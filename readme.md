## IP ALERT

### Your Selfhosting Companion 

IP Alert helps you alerting you early if your Server looses is know public IP Adress, that can happen if your Provide changes it. Be ready to responde to with the tool.


### Features 

Send a Mail to the Adminstor to Alert an Public IP Adress Change

### Launchflags

`py IPAlert.py --no-log` - disables logfolder logging
